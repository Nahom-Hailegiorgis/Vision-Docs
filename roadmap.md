# Vision Roadmap

## Short Term Goals
- Transition from snapshot to continuous detection mode
- Implement stereo and directional audio feedback
- Improve obstacle distance estimation accuracy

## Medium Term Goals
- Add customizable sound profiles for different users
- Optimize performance and reduce latency
- Develop accessibility-friendly UI enhancements

## Long Term Goals
- User testing with visually impaired individuals
- Open source select modules
- Explore integration with wearable devices
